  • 테이블을 생성한 후에 제약 조건을 추가하거나 제거할 필요성이 있음

    제약 조건도 테이블의 구조 정보에 속함으로 ALTER TABLE을 이용함

    추가

    ADD CONSTRAINT

    제거

    DROP CONSTRAINT

    제약 조건 이름을 지정해 두어야 추가나 제거가 쉬움

    추가

    ALTER TABLE 테이블명
    ADD [CONSTRAINT 제약조건명]
    제약조건 (속성명)

    제거

    ALTER TABLE 테이블명
    DROP CONSTRAINT 제약조건명
