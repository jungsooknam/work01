  • Q1
    MS-SQL에서 제공하는 기본 데이터 타입 중 숫자형 타입으로 자동증분이 되는 것은?
    • bigint사용자 체크
    • identity사용자 체크 정답 체크
    • char사용자 체크
    • small int사용자 체크
    정답 확인
  • 정답 2
    해설 자동 증가 속성을 가진 타입은 identity 타입입니다. bigint 타입은 64비트를 제공하는 -2^63~ 2^63-1(정수)를 나타내며, Smallint 타입은 16비트를 사용해서 -2^15~2^15-1(정수)를 나타냅니다. Char 타입은 문자 타입 중 하나로 숫자형 타입이 아닙니다.

    테이블의 구조를 변경하기 위한 구문을 고르시오.
    • ALTER TABLE 문사용자 체크 정답 체크
    • CREATE TABLE 문사용자 체크
    • DROP TABLE 문사용자 체크
    • TRUNCATE TABLE 문사용자 체크
    정답 확인
  • 정답 1
    해설 테이블의 구조를 변경하는 구문은 ALTER TABLE 문 입니다. CREATE TABLE 문은 테이블을 생성할 때 사용하는 구문이고, DROP TABLE 문은 테이블을 지울 때, TRUNCATE TABLE 문은 테이블의 모든 내용은 지우지만 테이블은 남기고자 할 때 사용하는 구문입니다.
